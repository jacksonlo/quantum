Quantum

Welcome to Quantum the simple Application Starter Kit. Just click below to perform the configuration and setup of Quantum. Remember that Quantum is a Starter kit that is built on EllisLab CodeIgniter so please bookmark this: link for thier user-guide.


Requirements

  • PHP 5.1.6+
  • MySQL

Steps

Please follow these steps prior to running the configuration.

  1. Please add a database to your server.
  2. Import the quantum.sql file to the database
  3. Update the config.json file with the details
  4. Then click the "Initialize Configuration" button below

* Please note *

Make sure you've set the details in your config.json file before you run the configuration.
